# TaskList

## O que é?

- É uma aplicação onde se pode cadastrar quadros de tarefas.

## Tecnologias utilizadas:

- Java 8
- JSF 2.2
- Primefaces 8
- Hibernate 4
- MySQL
- Tomcat 7
- Bootstrap 4
